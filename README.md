# InVivo AI Platform

An AI-ready data platform for modeling drug-induced liver toxicity using molecular features and machine learning.

This project demonstrates an end-to-end pipeline including:

• data ingestion  
• ETL pipelines  
• machine learning modeling  
• feature importance analysis  
• database integration (PostgreSQL – upcoming)

The goal is to simulate an AI-enabled pharmacology data platform similar to those used in modern drug discovery environments.
